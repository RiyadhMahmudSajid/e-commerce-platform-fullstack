# ShopGrid - Multi-Category E-commerce Platform
` ShopHub ` is a sophisticated and modern e-commerce ecosystem designed to provide a seamless shopping experience. It serves as a one-stop destination for various consumer needs, offering a wide range of categories including Fashion, Electronics, Home Appliances, and Lifestyle products.
The platform is built with a focus on High Performance, Scalability, and Security. Whether a user is just browsing for the latest trends or a registered member making a high-value purchase, ShopGrid ensures a smooth and intuitive journey from the landing page to the final payment confirmation.
# Role-Based Access Control (RBAC)
## 👤User Features ##
* Guest Browsing: Any visitor can browse products and add them to the Cart.
* Member Access: Users must register/log in to add products to their Wishlist and place Orders.
* Profile Management: Users can update their personal profiles easily.
* Secure Payment: Integrated with SSLCommerz for safe and easy payments.
* Review System: Users can leave reviews and ratings for the products they buy.
## 🛡️ Admin Features
* Product Management: Admin can Add, Edit, or Delete any product.
* Category Control: Admin can create and manage different product categories.
* Order Tracking: View all customer orders and update the Order Status.
* Review Monitoring: Admin can see all customer reviews for quality control.
#  🛠️ Tech Stack
* Frontend: React.js, Tailwind CSS, Axios.
* Data Fetching: TanStack Query (React Query) for efficient server-state management.
* Form Handling: React Hook Form for fast and flexible form validation.
* Backend: Node.js, Express.js.
* Database: MongoDB.
* Authentication: Firebase (Login) & JWT (Security).
* Payment: SSLCommerz.
# ✨ Key Functionalities</u>
* Role-Based Access: Different permissions and private dashboards for Admin and User.
* Secure Transactions: Integrated SSLCommerz payment gateway for a safe and seamless checkout experience.
* Optimized Data Fetching: Used TanStack Query for caching, synchronizing, and updating server state without extra reloading.
* Smooth Form Validation: Implemented React Hook Form to ensure error-free user registration and profile updates.
* JWT Security: Protected private routes and API endpoints using JSON Web Tokens.
* Wishlist & Cart: Dynamic shopping cart for all visitors and a persistent Wishlist for registered users.
* Responsive UI: A fully mobile-friendly design built with Tailwind CSS.
